# WealthPulse

A role-aware operational dashboard for wealth management — surfacing AI-driven
insights, portfolio data, and action items through a unified Client 360 interface.

🔗 [Live Site](https://protogen-301-operational-dashboard.vercel.app/)

## About

Women and men are told to "check your portfolio" and "talk to your advisor" — but
no one gives them a single view that actually connects the dots. WealthPulse gives
clients and advisors that view. It translates complex financial data into plain-language
insights tailored to each viewer's role, surfaces proactive alerts before issues
escalate, and prioritizes action items so nothing falls through the cracks.

## Features

• Role-Based Views — client and advisor personas see tailored information from the same underlying data
• AI Insight Feed — proactive alerts and recommendations surfaced before issues escalate
• Portfolio Snapshot — at-a-glance allocation and performance visualization
• Goal Tracker — visual progress toward financial milestones
• Action Items — prioritized tasks with status tracking
• Retirement Projections — forward-looking scenario modeling
• WCAG 2.1 AA Accessible — designed for inclusive use

## Tech Stack

• [Vue 3](https://vuejs.org/) (Composition API)
• [Vuetify 4](https://vuetifyjs.com/) for Material Design components
• [Vue Router 5](https://router.vuejs.org/) for client-side routing
• [Vite 8](https://vite.dev/) for build tooling
• TypeScript
• Vercel for hosting

## Getting Started

```bash
# Clone the repo
git clone https://github.com/emily-boggs/protogen-301-operational-dashboard.git
cd protogen-301-operational-dashboard

# Install dependencies
npm install

# Run development server
npm run dev
```
